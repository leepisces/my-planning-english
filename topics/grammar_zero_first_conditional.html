Zero & First Conditional

Introduction

Conditionals are sentence structures used to talk about possible or imaginary situations and their consequences. In this lesson, we'll focus on the Zero and First Conditional forms from Unit 29 (p.78) in Destination B1.

Zero Conditional

We use the Zero Conditional to talk about facts or things that are generally true, scientific facts, or things that always happen under certain conditions.

If + present simple, present simple

Examples:

If you heat water to 100°C, it boils.

This is a scientific fact that is always true.

If you don't water plants, they die.

This is a general truth that always happens under this condition.

Ice melts if you heat it.

Note that we can change the order of the clauses. The meaning stays the same.

Note: In the Zero Conditional, we can also use "when" instead of "if" with the same meaning.

When you heat water to 100°C, it boils.

First Conditional

We use the First Conditional to talk about real or possible situations in the future and their probable results.

If + present simple, will + infinitive

Examples:

If it rains tomorrow, I will stay at home.

This is a possible future situation (it might rain) and its probable result.

I'll help you with your homework if I have time.

Again, the order of clauses can be changed. When the "if clause" comes second, we don't use a comma.

If you study hard, you will pass the exam.

This expresses a real possibility in the future.

Note: In the First Conditional, we can also use other modal verbs instead of "will":

  • If it rains, I might cancel the picnic. (possibility)
  • If you finish early, you can go home. (permission)
  • If you want to improve, you should practice every day. (advice)

Comparing Zero and First Conditionals

Type Structure Use Example
Zero Conditional If + present simple, present simple General truths, scientific facts, things that always happen If you mix blue and yellow, you get green.
First Conditional If + present simple, will + infinitive Real or possible situations in the future If it rains tomorrow, we will cancel the match.

Common Mistakes

Incorrect: If it will rain tomorrow, I will stay at home.

Correct: If it rains tomorrow, I will stay at home.

We don't use "will" in the if-clause of a first conditional sentence.

Incorrect: When you will arrive, call me.

Correct: When you arrive, call me.

Similarly, we don't use "will" after "when" when referring to the future.

Exercise 1: Zero or First Conditional?

Identify whether each sentence is a Zero Conditional (Z) or First Conditional (F):

1. If you don't brush your teeth, they get cavities. _____
2. If it snows tonight, schools will close tomorrow. _____
3. Water freezes if the temperature falls below 0°C. _____
4. If you finish your work early, you can go home. _____
5. If you mix red and white, you get pink. _____

Exercise 2: Complete the sentences

Complete these First Conditional sentences:

1. If it rains tomorrow, we _____________ (cancel) the picnic.
2. I _____________ (help) you if you ask me nicely.
3. If she _____________ (study) hard, she will pass the exam.
4. We _____________ (be) late if we don't leave now.
5. If you _____________ (not/hurry), you will miss the train.

Exercise 3: Zero Conditional

Complete these Zero Conditional sentences:

1. If you _____________ (heat) ice, it melts.
2. Plants _____________ (die) if they don't get water.
3. If you _____________ (not/eat) for a long time, you get hungry.
4. She _____________ (feel) tired if she works too much.
5. If you _____________ (press) this button, the machine stops.
